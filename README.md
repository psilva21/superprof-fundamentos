# Atalhos vscode

| atalho | descrição |
|--------|-----------|
|ctrl + j| abre o terminal integrado|
|ctrl + p| pesquisar arquivos|
| ctrl + shift + n| abre uma nova janela do vscode|
| ctrl + n | cria um arquivo|
| ctrl + k + ctrl + o | abrir um projeto|
|ctrl + w| fechar janela ou instância|
